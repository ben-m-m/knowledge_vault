# Description

StudyVault is a responsive Book & Notes Vault application that allows students to organize books, notes, tags, reading progress, and highlights in a single interface.

It supports CRUD operations, regex-powered search, local storage persistence, JSON import/export, accessibility features, and responsive layouts.

---

## User Stories

- As a student, I want to add books so I can keep track of my reading collection.
- As a student, I want to write notes about books so I can remember important ideas.
- As a student, I want to search books and notes using keywords or regex patterns.
- As a student, I want my data to persist after refreshing the page.
- As a student, I want to export my library as JSON.

---

## Accessibility Plan

- Semantic landmarks
- Skip link
- Keyboard navigation
- Focus indicators
- ARIA live regions
- Accessible form labels
- WCAG-compliant color contrast
